# Roboto

A GitHub robot for doing chores on repos

> A GitHub App built with [Probot](https://github.com/probot/probot) that A robot for doing chores on repos

## Setup

```sh
# Install dependencies
npm install

# Compile
npm run build

# Run
npm run start
```
